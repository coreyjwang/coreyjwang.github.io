ME433 Inclinometer

Overview:

This was a class project for the class ME433, in which I wired and programmed a simple inclinometer, which is a device that measures slope. This device would read the slope and write the data to the LCD screen, which would then display the direction.

An image of the inclinometer in use is included to the right

Relevant tools/technologies/skills:

NU32 development board (PIC32 microcontroller), embedded C, MPLab X IDE, interfacing with LCD, SPI, I2C, breadboard.

NICO102 Data Project

This was a class project for the class NICO102, in which I analyzed some data taken from a Kaggle dataset. The dataset contained data from real-life soccer stats as well as some data taken from the FIFA game franchise. The class was an introduction to programming for big data using Python, so the project consisted primarily of using the Python libraries Pandas and Matplotlib to process, clean, and visualize the data from the dataset. Least squares regression was also done for some of the data.

An example of one of the generated plots is displayed to the right.

Tools/technologies:

Python, Matplotlib, data processing.

ME449 Mobile Manipulation

This was a class project for the class ME449: Robotic Manipulation, in which I wrote python code using the Modern Robotics library in order to plan the trajectory for the end-effector of a simulated youBot. The youBot is a robot with a mobile base and a 5R robot arm, which was simulated in the CoppeliaSim software for this class.

A video demo of the final project is shown to the right.

Tools/technologies:

Python, kinematics, odometry, trajectory planning, feedback control.
